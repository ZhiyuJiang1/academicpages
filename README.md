# Zhiyu Jiang Academic Website

This repository contains the source code for my academic website.

- Website: https://zhiyujiang1.github.io/academicpages/
- Repository: https://github.com/ZhiyuJiang1/academicpages

## Structure

- `_pages/`: main pages such as About, CV, Publications, Talks, Awards, and Teaching
- `_publications/`: publication entries
- `_talks/`: talks and presentations
- `_teaching/`: teaching records
- `_awards/`: awards and scholarships
- `_data/`: shared site data such as navigation
- `files/`: downloadable files such as PDFs and CVs
- `images/`: profile image and other static images
- `assets/`, `_includes/`, `_layouts/`, `_sass/`: theme and styling assets

## Notes

This site is built with Jekyll and based on the Academic Pages template, with local customization for my own academic profile.

Several template-only files and optional utilities that were not being used have been removed to keep the repository easier to maintain.
